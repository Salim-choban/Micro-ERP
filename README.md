# Micro ERP

## Description
Micro ERP is a simple ERP module developed using Odoo 18.
It provides basic management for:
- Partners (Customers & Suppliers)
- Products
- Sales Orders
- Purchase Orders

## Features
- Partner Management
- Product Management
- Sales Order Management
- Purchase Order Management
- Order State Workflow
- Automatic Total Calculation
- Data Validation

## Technologies Used
- Odoo 18
- Python
- XML
- PostgreSQL
- Docker

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Salim-choban/Micro-ERP.git
```

2. Copy the module to the Odoo addons folder.

3. Update Apps List.

4. Install **Micro ERP** module.
